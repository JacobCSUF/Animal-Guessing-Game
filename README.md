This is an animal guessing game where users can think of an animal and have an AI try to guess what animal they are thinking of

Code layout: main.py is the starting point of the application. It is a FastAPI server that opens the home file found in the "templates" folder

game_logic.py contains all game logic, including the animal vector data with 40 features per animal, generated with an LLM.

ai_question.py created any questions generated by an LLM.
